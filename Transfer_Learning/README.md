# 🔁 Transfer Learning

This folder focuses on leveraging pretrained models to improve performance and reduce training time.

## 📌 Project Included

### 🔹 Transfer Learning for Image Classification
- Used pretrained models (e.g., MobileNet / ResNet)
- Fine-tuned layers for custom dataset

## ⚙️ Tech Stack
- TensorFlow / Keras

## 📊 Key Features
- Faster training using pretrained weights
- Improved accuracy with limited data

## 🎯 Key Learnings
- Concept of feature reuse
- Fine-tuning vs feature extraction
- Importance of pretrained models

## 🚀 Impact
Transfer learning is widely used in industry, and this project demonstrates practical application in real-world scenarios.
