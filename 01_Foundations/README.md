# 🧠 Foundations of Machine Learning & Deep Learning

This folder contains implementations of core concepts that form the backbone of machine learning and deep learning.

## 📌 Topics Covered

### 🔹 Gradient Descent Variants
- Implemented Batch Gradient Descent (BGD), Stochastic Gradient Descent (SGD), and Mini-Batch Gradient Descent
- Compared convergence speed and stability

### 🔹 Gradient Descent Implementation
- Built optimization logic from scratch
- Visualized loss reduction over iterations

### 🔹 Neural Network from Scratch
- Implemented forward propagation and backpropagation manually
- Used NumPy to simulate training without deep learning libraries

## ⚙️ Tech Stack
- Python
- NumPy
- Matplotlib

## 🎯 Key Learnings
- Understanding how optimization works internally
- Trade-offs between different gradient descent techniques
- Deep understanding of neural network computations

## 🚀 Why This Matters
These implementations helped build strong fundamentals required to understand advanced deep learning models like CNNs, RNNs, and Transformers.
