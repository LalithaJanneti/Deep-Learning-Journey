# 🖼️ Convolutional Neural Networks (CNN)

This folder contains implementations of CNN models for image classification tasks.

## 📌 Projects Included

### 🔹 CIFAR-10 Image Classification
- Built a CNN model to classify images into 10 categories
- Implemented convolution, pooling, and dense layers

### 🔹 Handwritten Digit Recognition
- Trained CNN on digit datasets
- Achieved high accuracy in digit classification

### 🔹 Flower Classification with Data Augmentation
- Applied image augmentation techniques (flip, rotation, zoom)
- Improved model generalization on limited data

### 🔹 Handwritten Numbers Classification
- Implemented CNN for digit recognition tasks

## ⚙️ Tech Stack
- TensorFlow / Keras
- NumPy
- Matplotlib

## 📊 Key Features
- Feature extraction using convolution layers
- Handling image datasets
- Data augmentation techniques

## 🎯 Key Learnings
- How CNNs extract spatial features
- Importance of data augmentation
- Avoiding overfitting in image models

## 🚀 Impact
These projects demonstrate strong understanding of computer vision and deep learning techniques used in real-world applications.
